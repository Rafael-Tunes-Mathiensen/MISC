# 📚 Dev Knowledge Base

Repositório pessoal utilizado para armazenar **documentações, anotações, snippets de código, ferramentas e materiais úteis** para programação e desenvolvimento de software.

A ideia deste repositório é servir como uma **base de conhecimento centralizada**, facilitando consultas rápidas e reutilização de informações durante estudos e projetos.

```
📦 MISC

 ┣ 📂 snippets        # Trechos de código reutilizáveis
 ┣ 📂 configs         # Arquivos de configuração
```
